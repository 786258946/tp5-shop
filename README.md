## 使用方法：
前端页面比较简陋，可以自己装修~

1、运行 php 相关环境
2、导入数据库 cfshop 
3、修改 application/database.php 相关参数
4、地址：
> 前台：http://localhost/demo/index.php
> 后台：http://localhost/demo/login.php
5、后台登录账号密码都是：admin
